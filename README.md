# Supermart Grocery Sales Dashboard

Interactive dashboard built with Dash and Plotly for analyzing the Supermart Grocery Sales dataset.

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
cd dashboard
python app.py
```

Open http://127.0.0.1:8050 in your browser.

## Pages

- **Overview** — KPI cards, monthly sales/profit trend, yearly breakdown
- **Categories & Regions** — category profitability, treemap, regional comparison
- **Discounts & Performance** — discount impact analysis, top customers and cities
